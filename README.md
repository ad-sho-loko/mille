## mille 

mille is a toy text editor in less than 1K lines of code.
It aims to be simple, readable codes.

## Install

```
go get github.com/ad-sho-loko/mille
```

## Usage

```
./mille <filename>
```

## Keys

|  Key  |  Description  |
| ---- | ---- |
|  `Ctrl-H`  |  Backspace |
|  `Ctrl-A`  |  Move Caret to Line Start |
|  `Ctrl-E`  |  Move Caret to Line End |
|  `Ctrl-P`  |  Up |
|  `Ctrl-F`  |  Right |
|  `Ctrl-N`  |  Down |
|  `Ctrl-B`  |  Left |
|  `Ctrl-S`  |  Save |
